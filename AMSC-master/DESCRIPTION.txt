
Auto-McDonald's-Survey-Completer
======

An automatic survey completer for McDonald's receipt surveys.
It uses the survey code printed on every McDonald's receipt,
completes the survey and prints out the Validation Code.

The code is Python 3.

Installation
------------

Fast install:

::

    pip install amsc

For a manual install get this package:   

Example
--------

Input:

Code: 02699-13280-61922-14292-00021-2
Code: 02699-13920-62622-16314-00027-3
Code: 02699-13890-62022-11405-00021-2
